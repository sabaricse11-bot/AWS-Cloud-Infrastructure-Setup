## AWS Cloud Infrastructure Setup

## Project Overview
This project demonstrates the setup of a basic AWS cloud infrastructure using Amazon EC2, IAM, and Amazon S3. The objective is to gain hands-on experience with cloud infrastructure deployment, Linux server administration, and AWS security best practices.

## Features
- Launched and configured an Amazon EC2 Linux instance.
- Created IAM users and assigned appropriate permissions.
- Connected securely to the EC2 instance using SSH.
- Managed files using Amazon S3.
- Performed basic Linux administration tasks.

- AWS IAM
- Amazon S3
